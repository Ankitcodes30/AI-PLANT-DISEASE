# Data Directory

## Structure
- `raw/` - Original dataset (download from Kaggle)
- `processed/` - Preprocessed data

## Dataset Information
- **Name**: New Plant Diseases Dataset
- **Source**: Kaggle
- **Size**: 87,000+ images
- **Classes**: 38 plant disease categories

## Download Instructions
1. Visit Kaggle: https://www.kaggle.com/datasets/vipoooool/new-plant-diseases-dataset
2. Download the dataset
3. Extract to `raw/` directory
